Medicina Física e de Reabilitação

Início

NIHSS

( The National Institutes of Health Stroke Scale )

1A - Nível de consciência

+ 0

Alerta.

+ 1

Sonolento, mas despertável com mínima estimulação.

+ 2

Sonolento, requer estimulação repetida para responder.

+ 3

Coma.

1B - Orientação [ pergunte o mês e idade ]

+ 0

Responde a ambas questões corretamente.

+ 1

Responde a uma questão corretamente.

+ 2

Ambas respostas são incorrectas.

1C - Comandos [ Abrir e fechar os olhos, abrir e fechar a mão ]

+ 0

Realiza a ambas tarefas corretamente.

+ 1

Realiza uma tarefa corretamente.

+ 2

Não realiza nenhuma tarefa corretamente.

2. Olhar [ movimento ocular no plano horizontal ]

+ 0

Normal.

+ 1

Paralisia parcial do olhar.

+ 2

Desvio forçado do olhar.

3. Campos visuais

+ 0

Sem défices campimétricos.

+ 1

Hemianopsia parcial.

+ 2

Hemianopsia completa.

+ 3

Hemianopsia bilateral
(incluindo cegueira cortical).

4. Paralisia facial

+ 0

Movimentos simétricos e normais.

+ 1

Paralisia facial minor.
( apagamento de prega nasolabial, assimetria no sorriso )

+ 2

Paralisia facial central.
( região inferior da face )

+ 3

Paralisia facial central completa.
( ausência de movimentos faciais das regiões superior e inferior da face )

5A. Força muscular - Membro superior esquerdo

[ Posição: extensão dos braços com as palmas para baixo a 90º se sentado ou a 45º se deitado. É valorizada queda do braço se esta ocorre antes de 10 segundos. ]

+ 0

Sem queda.
Mantém o braço a 90º (ou 45º) por 10 segundos completos.

+ 1

Queda antes dos 10 segundos,
sem tocar na cama ou outro suporte.

+ 2

Alguma força contra a gravidade.
Não atinge ou não mantém a 90º (ou 45º)

+ 3

Sem força contra a gravidade.

+ 4

Sem movimento.

NT

Não testável.
( em caso de amputação ou fusão de articulação )

7. Ataxia dos membros

+ 0

Sem ataxia.

+ 1

Presente em 1 membro.

+ 2

Presente em 2 membros.

NT

8. Sensibilidade

+ 0

Normal.

+ 1

Diminuição da sensibilidade ligeira a moderada.

+ 2

Diminuição da sensibilidade grave a total.

+ 0
